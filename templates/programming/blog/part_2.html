Housekeeping:

Google's code prettify was implemented. Pretty indeed

CSS: I had gotten into the poor habit of using a bunch of inline styling for making slight tweaks. Many of these slight tweaks were the same or similar enough to merit their own file. All CSS inline styling was removed and added to its own 3rd css file now used in all HTML documents. Though small, this will serve as a building block for future personalization. Current, simple changes:

.indent {padding-left: 30px;}
.tab {text-indent: 30px;}

  float: left;
  position: relative;
  left: 50%;}
.nav_items{
  display: block;
  float: left;
  position: relative;
  right: 50%;

}

Image Hosting: I had a very haphazard collection of images on the site. Some pictures were hosted publicly on Dropbox, some were direct links from imgur while some others were hosted from Google drive. This was originally done simply because it was easier at the time and I honestly was simply trying to get anything to work, time to pay a little bit of technical debt and prevent it from getting too large. All images were moved to Amazon AWS

Broken JS: Very embarrassing: I realized all my Java script links were not working. Since the amount of working JS is quite low for this bootstrap theme, I didn't notice. The working code allows the nav bar to function while also applying some dynamic re-sizing for the pages, show the window's width change. 

script src="static/jquery.js">/script

Changed to:

script src="../../static/jquery.js">/script

Favicon: I got tired of looking at the default image within a tab: so I made my own. I took a picture of my dream bike, resized it to the correct combinations of dimensions. Lots of the lines get blurred when re-sizing the image down so much, so individual pixels were edited in photoshop. Annoying work, but, as bad as the image in the tab looks currently... it had looked quite a bit poorer.

Nav bar: The nav bar needed a few stylistic changes: It needed to be centered, its font was changed and letter shadow was added so that the test contrasts any background image better. Correct links were built and it basically looks not horrible.
